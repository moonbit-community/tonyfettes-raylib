# Dungeon Crawler 3D

A first-person roguelite dungeon crawler with procedurally generated floors, real-time combat, loot, spells, and boss fights.

## How to Play

Descend through procedurally generated dungeon floors. Fight enemies, collect loot, level up your character, and find the exit stairs on each floor. Every 5th floor features a boss encounter. Visit shops and shrines for permanent upgrades.

## Controls

| Key | Action |
|-----|--------|
| WASD | Move |
| Mouse | Look |
| Left Click / Space | Attack |
| 1-6 | Cast spell |
| E | Interact / Open chest |
| Tab | Toggle inventory |
| M | Toggle minimap |
| P | Character stats |
| Q | Use potion |
| Left Shift | Sprint |
| Escape | Pause |

## Build & Run

```bash
moon -C examples build --target native raylib_dungeon_crawler_3d/
moon -C examples run --target native raylib_dungeon_crawler_3d/
```
