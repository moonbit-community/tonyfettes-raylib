# raylib_neon_heist_extraction_2026

A top-down twin-stick stealth-action game where you infiltrate a neon-lit facility, hack terminals, loot vaults, fight guards, and extract with the goods.

## Build and Run

```bash
cd examples && moon build --target native raylib_neon_heist_extraction_2026/
cd examples && ./_build/native/debug/build/raylib_neon_heist_extraction_2026/raylib_neon_heist_extraction_2026.exe
```

## Controls

- **WASD / Arrow Keys**: Move player
- **J / Space**: Fire weapon (hold)
- **K**: Hack (hold near terminals/cameras)
- **L / Right Shift**: Dash
- **Enter**: Start game
- **F11**: Toggle borderless windowed

## How to Play

- Navigate through the facility, hacking security cameras and terminals
- Loot vaults and collect scattered items to meet the loot goal
- Fight or avoid patrolling guards -- use dash for quick evasion
- Deliver collected loot to the extraction zone to bank your score
- Manage HP, energy, and weapon heat while building kill combos
