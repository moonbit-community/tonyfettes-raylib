# Puzzle Dungeon 3D

An isometric 3D puzzle game where you push blocks, activate switches, and solve increasingly complex puzzles across 5 themed worlds.

## How to Play

Push blocks onto pressure plates to open gates and reach the exit. Each world introduces new mechanics: ice floors, conveyors, teleporters, mirrors, lasers, and more. Earn stars by solving puzzles in fewer moves.

## Controls

| Key | Action |
|-----|--------|
| Arrow Keys / WASD | Move player |
| Z | Undo last move |
| R | Restart level |
| Space / Enter | Confirm / Select |
| Escape | Pause / Back |
| Q | Quit |

## Build & Run

```bash
moon -C examples build --target native raylib_puzzle_dungeon_3d/
moon -C examples run --target native raylib_puzzle_dungeon_3d/
```
