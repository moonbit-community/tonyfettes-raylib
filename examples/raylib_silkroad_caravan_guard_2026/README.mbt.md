# Silkroad Caravan Guard 2026

A top-down action game where you guard a caravan of wagons traveling the Silk Road. Defend against waves of raiders approaching from multiple lanes while keeping the wagons intact.

## Build and Run

```bash
cd examples && moon build --target native raylib_silkroad_caravan_guard_2026/
cd examples && ./_build/native/debug/build/raylib_silkroad_caravan_guard_2026/raylib_silkroad_caravan_guard_2026.exe
```

## Controls

- **WASD / Arrow Keys**: Move the player guard
- **Mouse**: Aim direction
- **Left Mouse Button / Space**: Shoot
- **Shift**: Dodge roll
- **F11**: Toggle borderless windowed mode

## How to Play

- Protect 3 wagons from incoming raiders as the caravan advances
- Raiders come in different types and approach from multiple lanes
- Shoot raiders before they reach the wagons and deal damage
- Wagons slowly repair over time but can be destroyed if HP reaches zero
- Survive through all waves to achieve victory
