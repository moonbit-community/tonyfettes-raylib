# raylib_neon_rooftop_rush_2026

A neon-themed side-scrolling endless runner where you leap across rooftops, slide under obstacles, grapple to anchor points, and collect data chips across escalating stages.

## Build and Run

```bash
cd examples && moon build --target native raylib_neon_rooftop_rush_2026/
cd examples && ./_build/native/debug/build/raylib_neon_rooftop_rush_2026/raylib_neon_rooftop_rush_2026.exe
```

## Controls

- **Space / W / Up**: Jump (press again in air for double jump)
- **S / Down**: Slide (hold)
- **J / Mouse Click**: Grapple to anchor points
- **P**: Pause
- **R**: Restart

## How to Play

- Run automatically across procedurally generated neon rooftops
- Jump over vents and signs, slide under drones, grapple across gaps
- Collect glowing data chips to increase your score and build combos
- Clear stage distance goals for bonus points and increasing speed
- Manage health and stamina -- taking hits drains health, grappling costs stamina
