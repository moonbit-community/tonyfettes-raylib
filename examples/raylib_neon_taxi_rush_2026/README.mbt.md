# raylib_neon_taxi_rush_2026

A top-down neon taxi driving game where you pick up passengers, deliver them to destinations, collect fuel, and dodge traffic to earn fares and build combos.

## Build and Run

```bash
cd examples && moon build --target native raylib_neon_taxi_rush_2026/
cd examples && ./_build/native/debug/build/raylib_neon_taxi_rush_2026/raylib_neon_taxi_rush_2026.exe
```

## Controls

- **WASD / Arrow Keys**: Drive the taxi
- **J / Space**: Boost
- **K**: Horn (scatter nearby traffic)
- **L / Enter**: Pick up / drop off passengers
- **Enter**: Start game
- **F11**: Toggle borderless windowed

## How to Play

- Cruise the neon streets picking up waiting passengers shown as glowing markers
- Deliver passengers to their destination markers to earn fare cash
- Chain quick deliveries to build combos for bonus earnings
- Collect fuel pickups scattered on the road to keep driving
- Avoid collisions with traffic -- crashes drain HP and slow you down
- Run out of fuel or HP and the shift is over
