# raylib_ninja_rooftop_chase_2026

A side-scrolling action runner where a ninja sprints across rooftops, slashing enemies with a katana, throwing shurikens, and dashing through obstacles to reach a target distance.

## Build and Run

```bash
cd examples && moon build --target native raylib_ninja_rooftop_chase_2026/
cd examples && ./_build/native/debug/build/raylib_ninja_rooftop_chase_2026/raylib_ninja_rooftop_chase_2026.exe
```

## Controls

- **A/D or Left/Right**: Switch lanes
- **W / Up / Space**: Jump
- **S / Down**: Slide (hold)
- **J**: Slash attack
- **K**: Throw shuriken
- **L**: Dash
- **Enter**: Start game
- **F11**: Toggle borderless windowed

## How to Play

- Sprint automatically across rooftops in three lanes
- Slash nearby enemies with the katana for close-range kills
- Throw shurikens at distant enemies for ranged attacks
- Slide under overhead obstacles and jump over ground hazards
- Dash for a burst of invulnerable speed
- Build combos by chaining kills and coin pickups to reach the target distance
