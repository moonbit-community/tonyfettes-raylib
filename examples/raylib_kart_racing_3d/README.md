# Kart Racing 3D

An arcade kart racing game with 8 racers, drift boost mechanics, 8 power-up types, and 16 tracks across 4 themed cups.

## How to Play

Race against 7 AI karts across 4 cups. Drift through corners to build boost, collect item boxes for power-ups, and finish in the best position to earn cup points.

## Controls

| Key | Action |
|-----|--------|
| W / Up | Accelerate |
| S / Down | Brake / Reverse |
| A/D / Left/Right | Steer |
| Z / Left Shift | Drift (hold while turning) |
| Space | Use item |
| X | Look behind |
| Escape | Pause / Back |
| Q | Quit to menu |

## Build & Run

```bash
moon -C examples build --target native raylib_kart_racing_3d/
moon -C examples run --target native raylib_kart_racing_3d/
```
