# Skyhook Cargo Rescue 2026

A helicopter rescue game where you navigate through hazardous skies to pick up stranded survivors and deliver them to safety. Manage fuel, avoid storm clouds, drones, and ice hazards.

## Build and Run

```bash
cd examples && moon build --target native raylib_skyhook_cargo_rescue_2026/
cd examples && ./_build/native/debug/build/raylib_skyhook_cargo_rescue_2026/raylib_skyhook_cargo_rescue_2026.exe
```

## Controls

- **Mouse / Touch**: Click and hold to steer the helicopter toward the cursor
- **Keyboard**: Arrow keys or WASD for movement

## How to Play

- Fly your helicopter to pick up survivors scattered across the map
- Carry survivors back to the landing zone to score points
- Avoid hazards: storm clouds, hostile drones, and ice formations
- Collect pickups: fuel cans, medkits, and turbo boosts
- Manage your fuel gauge -- running out ends the mission
- Title screen, play, and results screens with touch/mouse support
