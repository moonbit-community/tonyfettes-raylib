# Platformer 3D

A 3D platformer across 4 themed worlds with 20+ levels. Jump across platforms, collect gems, avoid hazards, and race to the finish.

## How to Play

Navigate through levels by jumping across platforms, collecting gems and coins, and reaching the exit. Earn star ratings based on time, collectibles, and deaths. Unlock new worlds by collecting stars.

## Controls

| Key | Action |
|-----|--------|
| Arrow Keys | Move / Navigate menus |
| Space | Jump / Double jump |
| Left Shift | Dash |
| E | Interact |
| +/- | Zoom camera |
| M | Toggle minimap |
| Enter | Confirm |
| Escape | Pause / Back |
| Q | Quit to menu |

## Build & Run

```bash
moon -C examples build --target native raylib_platformer_3d/
moon -C examples run --target native raylib_platformer_3d/
```
