# raylib_wildfire_airlift_command_2026

A helicopter rescue and firefighting game. Fly over wildfire-threatened terrain to evacuate civilians from danger zones while dropping water on advancing fires. Manage fuel, hull integrity, water supply, and passenger capacity.

## Build and Run

```bash
cd examples && moon build --target native raylib_wildfire_airlift_command_2026/
cd examples && ./_build/native/debug/build/raylib_wildfire_airlift_command_2026/raylib_wildfire_airlift_command_2026.exe
```

## Controls

- **WASD/Arrows**: Fly helicopter
- **Space/J**: Pick up civilians / drop off at base / drop water on fires
- **R**: Restart mission
- Touch controls available via on-screen D-pad and action buttons

## How to Play

- Locate evacuation zones with stranded civilian groups across the map
- Fly close and press action to load survivors (limited capacity)
- Return to base to offload passengers and refuel/repair
- Drop water on fire zones to slow their spread and protect evacuation zones
- Avoid fire zones and obstacles that damage your hull
- Balance rescue missions with firefighting to complete the operation
