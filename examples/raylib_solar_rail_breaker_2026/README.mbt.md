# Solar Rail Breaker 2026

A circular arena shooter set on concentric orbital rings. Move between rings, dodge solar flares, and destroy waves of enemies that orbit around you in this unique radial combat game.

## Build and Run

```bash
cd examples && moon build --target native raylib_solar_rail_breaker_2026/
cd examples && ./_build/native/debug/build/raylib_solar_rail_breaker_2026/raylib_solar_rail_breaker_2026.exe
```

## Controls

- **A/D or Left/Right**: Rotate around the current ring
- **W/S or Up/Down**: Switch between rings (4 concentric rings)
- **Space / Left Mouse Button**: Fire bullets outward
- **Mouse / Touch**: On-screen touch controls supported

## How to Play

- Navigate 4 concentric orbital rings, fighting enemies that orbit at various speeds
- Enemies come in multiple types with different HP, fire patterns, and behaviors
- Dodge enemy bullets and solar flares that sweep across the rings
- Clear all enemies in a stage to advance to the next, with increasing difficulty
- Features title, play, stage clear, and game over screens
