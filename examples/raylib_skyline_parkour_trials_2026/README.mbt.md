# Skyline Parkour Trials 2026

A precision platformer with wall-jumping, coyote time, and timed stages. Race through obstacle courses, hit checkpoints for time bonuses, and reach the goal before time runs out.

## Build and Run

```bash
cd examples && moon build --target native raylib_skyline_parkour_trials_2026/
cd examples && ./_build/native/debug/build/raylib_skyline_parkour_trials_2026/raylib_skyline_parkour_trials_2026.exe
```

## Controls

- **A/D or Left/Right**: Move horizontally
- **Space / W / Up**: Jump
- **Mouse / Touch**: On-screen touch controls supported

## How to Play

- Navigate through multi-stage platforming courses with solid and hazard tiles
- Wall-jump by pressing jump while sliding against a wall
- Coyote time gives a brief window to jump after leaving a ledge
- Hit checkpoints to earn time bonuses and save progress within a stage
- Reach the goal zone to clear the stage and advance
- Falling below the stage or running out of time results in game over
- Features title, play, stage clear, game over, and victory screens
