# Space Combat 3D

A 6DOF space combat game with 12+ missions, multiple ship classes, 4 weapon types, and dogfighting AI.

## How to Play

Complete missions ranging from patrols to escorts to boss fights. Manage shields, hull, weapon energy, and ammo. Upgrade your loadout between missions. Lock on to enemies and lead your shots.

## Controls

| Key | Action |
|-----|--------|
| WASD | Pitch and yaw |
| Q/E | Roll left/right |
| Left Shift | Afterburner |
| Left Ctrl | Brake |
| Space | Fire primary weapon |
| 1-4 | Switch weapon type |
| T | Toggle target lock |
| Tab | Cycle targets |
| R | Reload / Restart |
| Escape | Pause / Back |

## Build & Run

```bash
moon -C examples build --target native raylib_space_combat_3d/
moon -C examples run --target native raylib_space_combat_3d/
```
