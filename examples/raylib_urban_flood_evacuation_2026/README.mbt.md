# raylib_urban_flood_evacuation_2026

A rescue boat simulation where you navigate rising floodwaters to evacuate stranded civilians. Manage fuel, hull integrity, and capacity while dodging flood cells and debris barriers to deliver survivors to shelters.

## Build and Run

```bash
cd examples && moon build --target native raylib_urban_flood_evacuation_2026/
cd examples && ./_build/native/debug/build/raylib_urban_flood_evacuation_2026/raylib_urban_flood_evacuation_2026.exe
```

## Controls

- **WASD/Arrows**: Steer the rescue boat
- **Space/J**: Rescue nearby civilians / drop off at shelter
- **R**: Restart mission
- Touch controls via on-screen D-pad and action buttons

## How to Play

- Navigate the flooded city to find stranded civilian groups
- Approach groups and press action to pick them up (limited capacity)
- Deliver survivors to active shelter zones for points
- Avoid flood cells and debris barriers that damage your hull
- Use the ping system to locate distant groups; manage fuel to avoid getting stranded
