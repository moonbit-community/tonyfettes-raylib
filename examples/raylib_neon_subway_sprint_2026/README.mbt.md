# raylib_neon_subway_sprint_2026

A three-lane endless runner set in neon subway tunnels where you dodge obstacles, jump over barriers, dash through breakable objects, and avoid incoming trains.

## Build and Run

```bash
cd examples && moon build --target native raylib_neon_subway_sprint_2026/
cd examples && ./_build/native/debug/build/raylib_neon_subway_sprint_2026/raylib_neon_subway_sprint_2026.exe
```

## Controls

- **W/S or Up/Down**: Switch lanes
- **K / Space**: Jump
- **J / Left Shift**: Dash
- **R**: Restart

## How to Play

- Sprint through neon-lit subway tunnels at ever-increasing speed
- Switch between three lanes to avoid obstacles and oncoming trains
- Jump over low barriers and dash through breakable obstacles
- Collect coins and power-ups to boost your score
- Watch for train warnings -- get out of that lane before the train arrives
- Survive as long as possible to achieve a high score
