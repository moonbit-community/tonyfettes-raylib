# Teahouse Midnight Service 2026

A fast-paced teahouse management game where you brew and serve five types of tea to impatient late-night customers while managing focus, reputation, and time.

## Build and Run

```bash
cd examples && moon build --target native raylib_teahouse_midnight_service_2026/
cd examples && ./_build/native/debug/build/raylib_teahouse_midnight_service_2026/raylib_teahouse_midnight_service_2026.exe
```

## Controls

- **WASD / Arrow Keys / D-pad**: Move the server
- **J / Interact button**: Pick up water/leaves, brew tea, serve customers
- **K (hold) / Focus button**: Enter focus mode (slows time, drains focus meter)
- **Space / Dash button**: Quick dash (costs focus)
- **Mouse/Touch**: On-screen buttons for mobile-style input

## How to Play

Run a midnight teahouse serving Jasmine, Oolong, Pu-erh, Matcha, and Berry teas. Collect water from the kettle, pick tea leaves, brew at the station, then deliver the correct tea to waiting customers. Build combos by serving quickly. Missed or wrong orders damage reputation. Clear each stage's score goal before time expires to advance.
