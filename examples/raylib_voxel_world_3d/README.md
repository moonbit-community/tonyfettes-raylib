# Voxel World 3D

A Minecraft-inspired voxel sandbox with block breaking/placing, crafting, day/night cycle, biomes, caves, and survival mechanics.

## How to Play

Explore a procedurally generated voxel world. Mine blocks, craft tools, build structures, and survive against hostile mobs that spawn at night. Discover different biomes with unique terrain and resources.

## Controls

| Key | Action |
|-----|--------|
| WASD | Move |
| Mouse | Look |
| Left Click | Break block / Attack |
| Right Click | Place block |
| 1-9 | Select hotbar slot |
| E | Open inventory |
| C | Open crafting |
| F | Toggle fly mode |
| G | Drop item |
| R | Cycle tool |
| Q | Quick craft |
| Left Shift | Sprint |
| Space | Jump |
| F3 | Toggle debug info |
| Escape | Pause / Back |

## Build & Run

```bash
moon -C examples build --target native raylib_voxel_world_3d/
moon -C examples run --target native raylib_voxel_world_3d/
```
