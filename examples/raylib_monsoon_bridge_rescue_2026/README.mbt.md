# raylib_monsoon_bridge_rescue_2026

A flood rescue action game where you pilot a boat through monsoon-swollen waters, dodging debris and lightning strikes to rescue survivors stranded on rafts and collapsing bridge spans.

## Build and Run

```bash
cd examples && moon build --target native raylib_monsoon_bridge_rescue_2026/
cd examples && ./_build/native/debug/build/raylib_monsoon_bridge_rescue_2026/raylib_monsoon_bridge_rescue_2026.exe
```

## Controls

- **WASD / Arrow Keys**: Steer the rescue boat
- **Space / J**: Boost engine
- **L / U**: Action (rescue survivors, pick up supplies)
- **Enter**: Start game
- **F11**: Toggle borderless windowed

## How to Play

- Navigate your rescue boat through turbulent flood currents and rising waters
- Approach stranded raft groups to rescue survivors and transport them to the evacuation camp
- Dock at supply stations to refuel and pick up rescue kits
- Avoid floating debris, lightning strikes, and collapsing bridge spans
- Manage hull integrity, fuel, and passenger capacity while racing against the storm
