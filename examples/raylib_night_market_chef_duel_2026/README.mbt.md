# raylib_night_market_chef_duel_2026

A fast-paced cooking management game where you prep, cook, and serve night market street food dishes against a timer, building combos for speed bonuses.

## Build and Run

```bash
cd examples && moon build --target native raylib_night_market_chef_duel_2026/
cd examples && ./_build/native/debug/build/raylib_night_market_chef_duel_2026/raylib_night_market_chef_duel_2026.exe
```

## Controls

- **WASD / Arrow Keys**: Navigate between cooking stations
- **J**: Select order / confirm action
- **K**: Cancel / discard
- **Space**: Speed boost (when charged)
- **P**: Pause
- **R**: Restart

## How to Play

- Incoming orders appear with a patience timer -- serve them before they expire
- Navigate between prep, grill, and plating stations to complete each dish
- Dishes include skewers, noodles, squid, and fried rice with different prep times
- Build combos by serving orders quickly in succession
- Balance speed vs. quality -- overcooked or undercooked dishes lower your score
- The pace increases as orders arrive faster over time
