# Tower Defense 3D

A 3D tower defense game with 8 tower types, tower upgrades, multiple enemy types, and strategic depth.

## How to Play

Place towers along enemy paths to stop waves of enemies from reaching your base. Earn gold from kills, upgrade towers, and manage your economy with interest income between waves.

## Controls

| Key | Action |
|-----|--------|
| Arrow Keys / WASD | Move camera / Navigate menus |
| 1-8 | Select tower type |
| Space / Enter | Place tower / Start wave |
| E | Upgrade selected tower |
| Q | Sell selected tower |
| F | Cycle targeting mode |
| M | Toggle map select |
| R | Restart |
| Escape | Pause / Back |

## Build & Run

```bash
moon -C examples build --target native raylib_tower_defense_3d/
moon -C examples run --target native raylib_tower_defense_3d/
```
