# Survival Arena 3D

A first-person wave-based survival shooter. Fight escalating waves of enemies with 6 weapons, buy upgrades between waves, and survive as long as you can.

## How to Play

Survive waves of enemies in a procedural arena. Kill enemies to earn money, then spend it at the between-wave shop on new weapons, upgrades, ammo, and health. Combo kills for score multipliers.

## Controls

| Key | Action |
|-----|--------|
| WASD | Move |
| Mouse | Look |
| Left Click | Fire |
| R | Reload |
| 1-6 | Switch weapon |
| Left Shift | Sprint |
| C | Crouch |
| Space | Jump / Buy (in shop) |
| Q | Quick switch |
| Escape | Pause |

## Build & Run

```bash
moon -C examples build --target native raylib_survival_arena_3d/
moon -C examples run --target native raylib_survival_arena_3d/
```
