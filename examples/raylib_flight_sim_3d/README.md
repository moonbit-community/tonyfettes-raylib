# Flight Simulator 3D

An arcade flight simulator with 15 missions, 4 aircraft types, procedural terrain, weather systems, and air/ground combat.

## How to Play

Fly missions ranging from training flights to combat sorties. Manage throttle, fuel, and weapons while navigating weather conditions. Complete objectives to earn scores and unlock harder missions.

## Controls

| Key | Action |
|-----|--------|
| W/S | Pitch up/down |
| A/D | Yaw left/right |
| Q/E | Roll left/right |
| Up/Down | Throttle up/down |
| Left Shift | Afterburner |
| Left Ctrl | Brake |
| Space | Fire weapon |
| C | Cycle camera mode |
| R | Restart mission |
| Escape | Pause / Back |

## Build & Run

```bash
moon -C examples build --target native raylib_flight_sim_3d/
moon -C examples run --target native raylib_flight_sim_3d/
```
