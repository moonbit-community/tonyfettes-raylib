# raylib_drone_show_director_2026

A drone choreography management game. As the director, move around the field commanding drones to form patterns, fulfill audience requests, manage battery and temperature, and dodge wind gusts -- all while putting on a spectacular firework show.

## Build and Run

```bash
cd examples && moon build --target native raylib_drone_show_director_2026/
cd examples && ./_build/native/debug/build/raylib_drone_show_director_2026/raylib_drone_show_director_2026.exe
```

## Controls

- **WASD / Arrow keys**: Move the director
- **J/Space**: Command action (sync drones, fulfill requests)
- **K**: Boost ability
- **R**: Restart

## How to Play

- Move around the field and command drones to form requested patterns
- Fulfill audience requests before their deadlines for rewards
- Wind gusts push drones off course; reposition them to maintain quality
- Manage drone batteries and temperatures
- Recharge drones at stations scattered around the field
- Higher pattern quality earns more points
